# HariboHappyWorld
[하리보(HARIBO) IP를 활용한 체험형 미디어 전시](https://haribohappyworldjeju.com/) 프로젝트로 여러 개의 미니게임과 인터랙티브 콘텐츠를 통해 관람객이 공간을 이동하며 체험하도록 구성된 전시입니다.
본인이 담당한 구역은 **하리보 비치타운**과 **하리보 제작 공정** 두 개의 구역으로 각 구역은 서로 다른 체험 목적과 인터랙션 방식을 가집니다.

<ul>
  <li>
    <b>하리보 비치타운</b>
    <ul>
      <li>미니게임을 중심으로 구성된 구역입니다. 각 콘텐츠 체험 후 제공되는 리워드 마크를 전용 앱으로 인식하여 빙고를 완성하는 방식의 순환형 체험 구조를 가집니다.</li>
      <li>전체 구성: 미니게임 6종 + 체험 콘텐츠 2종</li>
      <li>담당 범위: 미니게임 3종 + 체험 콘텐츠 1종</li>
    </ul>
  </li>
  
  <li>
    <b>제작공정</b>
    <ul>
      <li>하리보 젤리의 제작 공정을 게임 형태로 체험할 수 있는 구역입니다. 버튼, 발판 등 외부 입력 장치를 활용한 물리적 인터랙션이 중심이며 명확한 보상이나 목표보다는 자유롭게 만지고 반응을 즐기는 체험에 초점을 둡니다.</li>
      <li>전체 구성: 콘텐츠 4종</li>
      <li>담당 범위: 콘텐츠 3종</li>
    </ul>
  </li>
</ul>
